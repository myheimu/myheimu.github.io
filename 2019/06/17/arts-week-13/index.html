Algorithm: LeetCode #34. Find First and Last Position of Element in Sorted Array

https://leetcode.com/problems/find-first-and-last-position-of-element-in-sorted-array/

问题：从一个非严格递增数组中找到target目标值最先和最后出现位置

解答：分两次进行二分查找，分别寻找最先出现位置和最后出现位置，注意在可能重复的情况下的二分判断条件。

Runtime: 8 ms , faster than 92.79% of C++ online submissions for Find First and Last Position of Element in Sorted Array.
Memory Usage: 10.1 MB , less than 98.86% of C++ online submissions for Find First and Last Position of Element in Sorted Array.

Review: 《Interviewing for Google AI Residency, A Kaggle Gold Finish, DSNet Launch, Finishing my Undergraduate Degree》

https://medium.com/@init_27/interviewing-for-google-ai-residency-a-kaggle-gold-finish-dsnet-launch-c621930b043d

这篇文章讲述了作者在读书过程中，一边读书、一边学习AI的经历。

作者在本科四年生涯中，白天完成计算机专业的学习工作，晚上则学习机器学习、当自由职业者、写博客、组织各类学术聚会。在最后一年，则跟随导师完成了“使用深度学习与NLP生成音乐”的论文项目。

作者参与构造了一个叫DSNet的学术交流组织，用来聚集数据科学初学者，分享PyTorch、Fast.ai和其他内容，已经有几千人的参与规模。

作者在Kaggle竞赛平台上参与了很久，通过组队开发，拿到了不少金牌，最终在竞赛、核心、讨论等几个方面都进入了Top 1%。

随后作者参与了2019年Google AI的实习项目面试，用了几个月的时间准备数据结构和编程题，也阅读了很多论文，完成了fast.ai课程第二部分。在准备面试过程中，也参照了相关FAQ阅读了很多内容：

  • HackerRank’s 30 days of code challenge
  • Data Structures and Algorithms in Python by Michael T. Goodrich (book)
  • Cracking the Coding Interview by Gayle Laakmann McDowell (book)
  • Chris Olah’s blog
  • Ian Goodfellow’s Deep Learning textbook (Initial Chapters)

随后作者在面试过程中见到了很多让人惊喜的场景和人，但并没有通过面试。

fastai团队给予了作者非常多的帮助，包括AI知识、写博客的指导、学术交流组织的建设、Kaggle竞赛的指导。

最后作者列了未来的计划，包括重启博客计划、为学术交流组织继续贡献、拿到Kaggle Master、每周分享一篇论文摘要、每周总结Kaggle竞赛情况等等。
